## Technologies

Built with React.

## Installation

1. Clone the repository:

```
git clone https://github.com/LorenaSTN/List-of-countries.git
```

2. Navigate to the folder

```
cd List-of-countries
```

3. Install the dependencies:

```
npm install
```

4. Run the app

```
npm run dev
```
